# **App Name**: GoldenBarber

## Core Features:

- Google Sign-In: Implement 'Sign In with Google' (Google Identity Services) for user authentication, using Google OAuth.
- Profile Management: Users can manage their profiles.
- Appointment Scheduling: Allow users to book appointments with available barbers and time slots.
- Barber Profiles: Showcase barber profiles with details such as specialization, experience, and customer reviews.
- Service Catalog: Display services offered, along with pricing.

## Style Guidelines:

- Primary color: Deep gold (#D4AF37) to convey luxury and classic barbering.
- Background color: Desaturated light gold (#F5F5DC). Provides a subtle backdrop that doesn't detract from the content.
- Accent color: Deep brown (#8B4513). Enhances the classic and sophisticated barber aesthetic.
- Body and headline font: 'Alegreya', a serif font. To match the content, a classic serif like Alegreya is great for body and headline
